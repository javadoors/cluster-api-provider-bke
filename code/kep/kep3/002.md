
# BKE 分阶段组件版本升级完整设计方案
基于对 `javadoors/cluster-api-provider-bke` 仓库的分析，我提供一份完整的、不依赖任何 .md 文档的设计方案。
## 一、核心架构设计
### 1.1 分层模型
```
┌──────────────────────────────────────────────────────────────────┐
│                  BKECluster Controller (调和层)                  │
│  ├─ 监听 BKECluster 变化                                         │
│  ├─ 触发 Phase 执行                                              │
│  └─ 收集 Phase 执行结果                                          │
└──────────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────────┐
│            Phase Executor (编排层 - 无业务代码)                  │
│  ├─ 加载 ComponentVersion Manifest                               │
│  ├─ 比较版本差异 (old vs new)                                    │
│  ├─ 生成 Task 序列                                               │
│  └─ 监控 Task 执行进度                                           │
└──────────────────────────────────────────────────────────────────┘
                              ↓
┌───────────────────────────────────────────────────────────────────┐
│           Component Manifest (声明层 - YAML定义)                  │
│  ├─ ComponentVersion: 版本元信息                                  │
│  ├─ ComponentLifecycleManifest: 生命周期逻辑                      │
│  ├─ CompatibilityMatrix: 兼容性矩阵                               │
│  └─ UpgradePath: 升级路径策略                                     │
└───────────────────────────────────────────────────────────────────┘
                              ↓
┌───────────────────────────────────────────────────────────────────┐
│              Task Executor (执行层 - 通用引擎)                    │
│  ├─ Shell Task: 脚本执行                                          │
│  ├─ Builtin Plugin: 内置插件路由                                  │
│  └─ K8s Resource: Kubernetes资源操作                              │
└───────────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────────┐
│               基础设施层 (节点、操作系统、服务)                  │
└──────────────────────────────────────────────────────────────────┘
```
### 1.2 设计核心原则
| 原则 | 含义 | 实现方式 |
|------|------|---------|
| **声明式优于命令式** | 所有逻辑在YAML中，代码只调和 | ComponentVersion + Manifest CRD |
| **版本即一等公民** | 每个组件独立版本化 | 版本包含所有元信息 |
| **兼容性前置校验** | 升级前检查依赖 | CompatibilityMatrix CRD |
| **有状态支持** | etcd等有状态组件用状态机 | StatefulUpgrade spec |
| **容错与回滚内置** | 每步都有失败处理 | OnFailure + Rollback策略 |
| **幂等操作** | 所有步骤可重入 | 通过版本比对+条件判断 |
| **模块化任务执行** | 3种执行方式 | shell/builtin/k8s |

### 2.1 ComponentVersion CRD
```go
// api/capbke/v1beta1/componentversion_types.go
package v1beta1

import (
	metav1 "k8s.io/apimachinery/pkg/apis/meta/v1"
)

// ComponentVersionSpec 定义组件版本信息
type ComponentVersionSpec struct {
	// Component 是组件名称 (kubernetes, etcd, containerd, cni 等)
	Component string `json:"component"`
	
	// Version 是语义化版本号
	Version string `json:"version"`
	
	// VersionInfo 包含详细的子版本信息
	VersionInfo map[string]string `json:"versionInfo,omitempty"`
	
	// Compatibility 定义兼容性要求
	Compatibility ComponentCompatibility `json:"compatibility,omitempty"`
	
	// Lifecycle 定义生命周期管理
	Lifecycle ComponentLifecycle `json:"lifecycle"`
	
	// Artifacts 包含下载链接和校验和
	Artifacts []Artifact `json:"artifacts,omitempty"`
	
	// Status 表示版本状态 (Active/Deprecated/EOL)
	Status ComponentStatus `json:"status,omitempty"`
}

type ComponentCompatibility struct {
	// Dependencies 列出依赖的其他组件版本范围
	Dependencies []DependencySpec `json:"dependencies,omitempty"`
	
	// SupportedOSes 支持的操作系统列表
	SupportedOSes []string `json:"supportedOses,omitempty"`
	
	// SupportedArch 支持的CPU架构
	SupportedArch []string `json:"supportedArch,omitempty"`
	
	// BreakingChanges 破坏性变更说明
	BreakingChanges []string `json:"breakingChanges,omitempty"`
}

type DependencySpec struct {
	Component string `json:"component"`
	MinVersion string `json:"minVersion"`
	MaxVersion string `json:"maxVersion"`
}

type ComponentLifecycle struct {
	// Install 安装规格 (引用 ComponentLifecycleManifest.metadata.name)
	Install LifecycleStepRef `json:"install"`
	
	// Uninstall 卸载规格
	Uninstall LifecycleStepRef `json:"uninstall,omitempty"`
	
	// Upgrade 升级规格 (可以引用版本特定的 manifest)
	Upgrade LifecycleStepRef `json:"upgrade,omitempty"`
	
	// Verify 校验规格
	Verify LifecycleStepRef `json:"verify,omitempty"`
	
	// HealthCheck 健康检查规格
	HealthCheck HealthCheckSpec `json:"healthCheck,omitempty"`
}

type LifecycleStepRef struct {
	// Ref 引用 ComponentLifecycleManifest 的名称
	Ref string `json:"ref"`
	
	// Timeout 操作超时时间
	Timeout string `json:"timeout,omitempty"`
	
	// Retries 重试次数
	Retries int `json:"retries,omitempty"`
	
	// OnFailure 失败时的行为 (Fail/SkipAndContinue/Rollback)
	OnFailure string `json:"onFailure,omitempty"`
}

type HealthCheckSpec struct {
	// Enabled 是否启用健康检查
	Enabled bool `json:"enabled"`
	
	// Interval 检查间隔
	Interval string `json:"interval,omitempty"`
	
	// Timeout 检查超时
	Timeout string `json:"timeout,omitempty"`
	
	// Steps 检查步骤列表
	Steps []HealthCheckStep `json:"steps,omitempty"`
}

type HealthCheckStep struct {
	Name string `json:"name"`
	Type string `json:"type"` // shell / http / kubernetes
	Command string `json:"command,omitempty"` // 对于 shell 类型
	Endpoint string `json:"endpoint,omitempty"` // 对于 http 类型
	Selector string `json:"selector,omitempty"` // 对于 kubernetes 类型
}

type Artifact struct {
	Name string `json:"name"`
	Type string `json:"type"` // binary / container-image / bundle
	URL string `json:"url"`
	SHA256 string `json:"sha256"`
	Targets []string `json:"targets"` // master / worker / etcd
}

type ComponentStatus string

const (
	ComponentActive ComponentStatus = "Active"
	ComponentDeprecated ComponentStatus = "Deprecated"
	ComponentEOL ComponentStatus = "EOL"
)

// ComponentVersion is the Schema for ComponentVersion API
type ComponentVersion struct {
	metav1.TypeMeta   `json:",inline"`
	metav1.ObjectMeta `json:"metadata,omitempty"`
	Spec   ComponentVersionSpec   `json:"spec,omitempty"`
	Status ComponentVersionStatus `json:"status,omitempty"`
}

type ComponentVersionStatus struct {
	Phase string `json:"phase,omitempty"`
	LastUpdateTime *metav1.Time `json:"lastUpdateTime,omitempty"`
}
```

```go
// pkg/phaseframe/phases/ensure_nodes_env_refactor.go
package phases

import (
	"context"
	"fmt"
	"time"

	"github.com/go-logr/logr"
	ctrl "sigs.k8s.io/controller-runtime"

	bkev1beta1 "gopkg.openfuyao.cn/cluster-api-provider-bke/api/capbke/v1beta1"
	confv1beta1 "gopkg.openfuyao.cn/cluster-api-provider-bke/api/bkecommon/v1beta1"
	"gopkg.openfuyao.cn/cluster-api-provider-bke/pkg/phaseframe"
	"gopkg.openfuyao.cn/cluster-api-provider-bke/pkg/component"
)

// EnsureNodesEnv - 节点环境初始化阶段（声明式版本）
type EnsureNodesEnvRefactor struct {
	phaseframe.BasePhase
	componentMgr *component.Manager
	taskExecutor component.TaskExecutor
}

func NewEnsureNodesEnvRefactor(ctx *phaseframe.PhaseContext) phaseframe.Phase {
	return &EnsureNodesEnvRefactor{
		BasePhase: phaseframe.NewBasePhase(
			ctx,
			confv1beta1.BKEClusterPhase("EnsureNodesEnv"),
		),
		componentMgr: component.NewManager(ctx.Client),
		taskExecutor: component.NewDefaultTaskExecutor(ctx),
	}
}

// NeedExecute 判断是否需要执行 - 基于版本比对
func (e *EnsureNodesEnvRefactor) NeedExecute(old, new *bkev1beta1.BKECluster) bool {
	if !e.DefaultNeedExecute(old, new) {
		return false
	}

	// 如果是首次创建
	if old == nil {
		return true
	}

	// 比较节点环境相关的版本是否变化
	oldEnvVersion := e.getNodeEnvVersion(old)
	newEnvVersion := e.getNodeEnvVersion(new)

	return oldEnvVersion != newEnvVersion
}

// Execute 执行阶段
func (e *EnsureNodesEnvRefactor) Execute() (ctrl.Result, error) {
	ctx, _, cluster, log := e.Ctx.Untie()

	// Step 1: 前置处理
	if err := e.ExecutePreHook(); err != nil {
		return ctrl.Result{}, err
	}

	// Step 2: 执行核心逻辑
	result, err := e.executeNodeEnv(ctx, cluster, log)

	// Step 3: 后置处理
	if postErr := e.ExecutePostHook(err); postErr != nil {
		return ctrl.Result{}, fmt.Errorf("post hook failed: %w", postErr)
	}

	return result, err
}

// 核心执行逻辑
func (e *EnsureNodesEnvRefactor) executeNodeEnv(
	ctx context.Context,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (ctrl.Result, error) {
	log.Info("Starting EnsureNodesEnv phase",
		"phase", e.Name(),
		"version", e.getNodeEnvVersion(cluster))

	// 加载组件版本 Manifest
	nodeEnvVersion := e.getNodeEnvVersion(cluster)
	manifest, err := e.componentMgr.GetComponentVersionManifest(
		ctx,
		"node-environment",
		nodeEnvVersion,
	)
	if err != nil {
		return ctrl.Result{}, fmt.Errorf("load component manifest: %w", err)
	}

	// 前置条件检查
	if err := e.validatePreConditions(ctx, cluster, manifest); err != nil {
		return ctrl.Result{RequeueAfter: 10 * time.Second}, err
	}

	// 获取所有节点
	nodes, err := e.Ctx.NodeFetcher().GetBKENodesWrapperForCluster(ctx, cluster)
	if err != nil {
		return ctrl.Result{}, fmt.Errorf("fetch nodes: %w", err)
	}

	// 获取当前节点环境版本
	currentVersions := e.getCurrentNodeEnvVersions(nodes)

	// 生成升级任务
	tasks := e.componentMgr.GenerateTasks(
		manifest,
		currentVersions,
		nodeEnvVersion,
		nodes,
	)

	// 执行任务序列
	if err := e.executeTasks(ctx, tasks, cluster, log); err != nil {
		return ctrl.Result{RequeueAfter: 10 * time.Second}, err
	}

	// 验证执行结果
	if err := e.verifyExecution(ctx, manifest, cluster, log); err != nil {
		return ctrl.Result{RequeueAfter: 10 * time.Second}, err
	}

	log.Info("EnsureNodesEnv phase completed successfully")
	return ctrl.Result{}, nil
}

// 验证前置条件
func (e *EnsureNodesEnvRefactor) validatePreConditions(
	ctx context.Context,
	cluster *bkev1beta1.BKECluster,
	manifest *component.ComponentVersionManifest,
) error {
	for _, condition := range manifest.Spec.PreConditions {
		switch condition.Type {
		case "clusterHealthy":
			// 检查集群是否健康
			if !cluster.Status.Ready {
				return fmt.Errorf("cluster not healthy")
			}
		case "nodesReady":
			// 检查所有节点是否就绪
			nodes, _ := e.Ctx.NodeFetcher().GetBKENodesWrapperForCluster(ctx, cluster)
			for _, node := range nodes.Items {
				if !node.Status.Ready {
					return fmt.Errorf("node %s not ready", node.Name)
				}
			}
		}
	}
	return nil
}

// 执行任务序列
func (e *EnsureNodesEnvRefactor) executeTasks(
	ctx context.Context,
	tasks []component.Task,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) error {
	for i, task := range tasks {
		log.Info("Executing task",
			"index", i+1,
			"total", len(tasks),
			"name", task.Name,
			"type", task.Type)

		// 执行单个任务
		result, err := e.taskExecutor.Execute(ctx, task, &component.TaskContext{
			Cluster: cluster,
		})

		if err != nil {
			if task.OnFailure == "Fail" {
				return fmt.Errorf("task %s failed: %w", task.Name, err)
			}
			log.Error(err, "Task failed but continuing", "name", task.Name)
			continue
		}

		// 如果是异步任务，需要等待完���
		if result.IsAsync {
			if err := e.waitForTaskCompletion(ctx, result.TaskID, task.Timeout); err != nil {
				return fmt.Errorf("wait for task: %w", err)
			}
		}

		log.Info("Task completed successfully", "name", task.Name)
	}

	return nil
}

// 验证执行结果
func (e *EnsureNodesEnvRefactor) verifyExecution(
	ctx context.Context,
	manifest *component.ComponentVersionManifest,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) error {
	// 执行健康检查
	healthCheckSpec := manifest.Spec.Lifecycle.HealthCheck
	
	for _, check := range healthCheckSpec.Steps {
		log.Info("Running health check", "name", check.Name)

		result, err := e.taskExecutor.ExecuteHealthCheck(ctx, check)
		if err != nil {
			return fmt.Errorf("health check failed: %w", err)
		}

		if !result.Success {
			return fmt.Errorf("health check %s failed: %s", check.Name, result.Message)
		}
	}

	// 更新集群状态
	cluster.Status.Phase = confv1beta1.BKEClusterPhase("NodesEnvReady")
	
	return nil
}

// 等待任务完成
func (e *EnsureNodesEnvRefactor) waitForTaskCompletion(
	ctx context.Context,
	taskID string,
	timeout time.Duration,
) error {
	// 从 Command 或 Job 状态检查任务是否完成
	// 实现省略
	return nil
}

// 获取节点环境版本
func (e *EnsureNodesEnvRefactor) getNodeEnvVersion(

	// 组合多个组件版本作为节点环境版本
```

### 3.2 EnsureCerts Phase 整改
```go
// pkg/phaseframe/phases/ensure_certs_refactor.go
package phases

type EnsureCertsRefactor struct {
	phaseframe.BasePhase
	componentMgr *component.Manager
	taskExecutor component.TaskExecutor
}

func NewEnsureCertsRefactor(ctx *phaseframe.PhaseContext) phaseframe.Phase {
	return &EnsureCertsRefactor{
		BasePhase: phaseframe.NewBasePhase(
			ctx,
			confv1beta1.BKEClusterPhase("EnsureCerts"),
		),
		componentMgr: component.NewManager(ctx.Client),
		taskExecutor: component.NewDefaultTaskExecutor(ctx),
	}
}

func (e *EnsureCertsRefactor) NeedExecute(old, new *bkev1beta1.BKECluster) bool {
	if !e.DefaultNeedExecute(old, new) {
		return false
	}

	// 首次创建时需要生成证书
	if old == nil {
		return true
	}

	// 证书过期即将更新时重新生成
	// 或者 K8s 版本变化时需要重新配置
	return e.certificatesNeedRenew(new) ||
		e.getK8sVersion(old) != e.getK8sVersion(new)
}

func (e *EnsureCertsRefactor) Execute() (ctrl.Result, error) {
	ctx, _, cluster, log := e.Ctx.Untie()

	if err := e.ExecutePreHook(); err != nil {
		return ctrl.Result{}, err
	}

	result, err := e.executeCertificates(ctx, cluster, log)

	if postErr := e.ExecutePostHook(err); postErr != nil {
		return ctrl.Result{}, postErr
	}

	return result, err
}

func (e *EnsureCertsRefactor) executeCertificates(
	ctx context.Context,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (ctrl.Result, error) {
	// 加载证书管理 Manifest
	manifest, err := e.componentMgr.GetComponentVersionManifest(
		ctx,
		"certificates",
		cluster.Spec.ClusterConfig.Cluster.KubernetesVersion,
	)
	if err != nil {
		return ctrl.Result{}, err
	}

	if err := e.executeCertTasks(ctx, tasks, cluster, log); err != nil {
		return ctrl.Result{RequeueAfter: 10 * time.Second}, err
	}
```

### 4.1 Etcd 升级状态机
```go
// pkg/phaseframe/phases/ensure_etcd_upgrade_refactor.go
package phases

type EtcdUpgradeState string

const (
	EtcdStateCheckHealth    EtcdUpgradeState = "CheckHealth"
	EtcdStateBackup         EtcdUpgradeState = "Backup"
	EtcdStateUpgradeMember  EtcdUpgradeState = "UpgradeMember"
	EtcdStateVerifyMember   EtcdUpgradeState = "VerifyMember"
	EtcdStateAllUpgraded    EtcdUpgradeState = "AllUpgraded"
	EtcdStateVerifyCluster  EtcdUpgradeState = "VerifyCluster"
	EtcdStateCompleted      EtcdUpgradeState = "Completed"
	EtcdStateRollingBack    EtcdUpgradeState = "RollingBack"
	EtcdStateFailed         EtcdUpgradeState = "Failed"
)

type EnsureEtcdUpgradeRefactor struct {
	phaseframe.BasePhase
	componentMgr    *component.Manager
	taskExecutor    component.TaskExecutor
	etcdStateMgr    *EtcdStateManager
}

type EtcdUpgradeStatus struct {
	State             EtcdUpgradeState
	SourceVersion     string
	TargetVersion     string
	BackupPath        string
	CurrentMemberIdx  int
	TotalMembers      int
	MemberStatuses    map[int]string
	Error             string
	LastTransition    time.Time
}

func (e *EnsureEtcdUpgradeRefactor) Execute() (ctrl.Result, error) {
	ctx, _, cluster, log := e.Ctx.Untie()

	if err := e.ExecutePreHook(); err != nil {
		return ctrl.Result{}, err
	}

	// 加载当前升级状态
	status := e.etcdStateMgr.LoadStatus(cluster)

	// 状态机驱动升级
	nextState, err := e.transitionState(ctx, status, cluster, log)
	if err != nil {
		if status.State != EtcdStateRollingBack {
			status.State = EtcdStateRollingBack
			status.Error = err.Error()
			e.etcdStateMgr.SaveStatus(cluster, status)
		}
		return ctrl.Result{RequeueAfter: 5 * time.Second}, nil
	}

	// 保存状态
	if nextState != status.State {
		status.State = nextState
		status.LastTransition = time.Now()
		e.etcdStateMgr.SaveStatus(cluster, status)

		if nextState == EtcdStateCompleted {
			if postErr := e.ExecutePostHook(nil); postErr != nil {
				return ctrl.Result{}, postErr
			}
			return ctrl.Result{}, nil
		}

		return ctrl.Result{RequeueAfter: 10 * time.Second}, nil
	}

	return ctrl.Result{RequeueAfter: 10 * time.Second}, nil
}

func (e *EnsureEtcdUpgradeRefactor) transitionState(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	switch status.State {
	case EtcdStateCheckHealth:
		return e.handleCheckHealth(ctx, status, cluster, log)
	case EtcdStateBackup:
		return e.handleBackup(ctx, status, cluster, log)
	case EtcdStateUpgradeMember:
		return e.handleUpgradeMember(ctx, status, cluster, log)
	case EtcdStateVerifyMember:
		return e.handleVerifyMember(ctx, status, cluster, log)
	case EtcdStateAllUpgraded:
		return EtcdStateVerifyCluster, nil
	case EtcdStateVerifyCluster:
		return e.handleVerifyCluster(ctx, status, cluster, log)
	case EtcdStateCompleted:
		return EtcdStateCompleted, nil
	case EtcdStateRollingBack:
		return e.handleRollback(ctx, status, cluster, log)
	case EtcdStateFailed:
		return EtcdStateFailed, nil
	default:
		return status.State, fmt.Errorf("unknown state: %s", status.State)
	}
}

func (e *EnsureEtcdUpgradeRefactor) handleCheckHealth(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Info("Checking etcd cluster health before upgrade")

	// 加载 manifest
	manifest, _ := e.componentMgr.GetComponentVersionManifest(
		ctx,
		"etcd",
		cluster.Spec.ClusterConfig.Cluster.EtcdVersion,
	)

	// 执行健康检查任务
	checkTasks := e.componentMgr.GetHealthCheckTasks(manifest)

	for _, task := range checkTasks {
		_, err := e.taskExecutor.Execute(ctx, task, nil)
		if err != nil {
			return EtcdStateCheckHealth, fmt.Errorf("health check failed: %w", err)
		}
	}

	// 初始化升级状态
	status.SourceVersion = "3.5.8"  // 从集群状态读取
	status.TargetVersion = cluster.Spec.ClusterConfig.Cluster.EtcdVersion
	status.CurrentMemberIdx = 0
	status.TotalMembers = 3  // 从 etcd 集群读取

	log.Info("Etcd cluster is healthy", "members", status.TotalMembers)
	return EtcdStateBackup, nil
}

func (e *EnsureEtcdUpgradeRefactor) handleBackup(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Info("Creating etcd backup before upgrade")

	manifest, _ := e.componentMgr.GetComponentVersionManifest(
		ctx,
		"etcd",
		status.TargetVersion,
	)

	backupTasks := e.componentMgr.GetBackupTasks(manifest, status)

	for _, task := range backupTasks {
		result, err := e.taskExecutor.Execute(ctx, task, nil)
		if err != nil {
			return EtcdStateBackup, err
		}
		
		// 从执行结果获取备份路径
		if backupPath, ok := result.Output["backup_path"]; ok {
			status.BackupPath = backupPath.(string)
		}
	}

	log.Info("Backup completed", "path", status.BackupPath)
	return EtcdStateUpgradeMember, nil
}

func (e *EnsureEtcdUpgradeRefactor) handleUpgradeMember(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	if status.CurrentMemberIdx >= status.TotalMembers {
		return EtcdStateAllUpgraded, nil
	}

	memberIdx := status.CurrentMemberIdx
	log.Info("Upgrading etcd member",
		"index", memberIdx,
		"total", status.TotalMembers)

	manifest, _ := e.componentMgr.GetComponentVersionManifest(
		ctx,
		"etcd",
		status.TargetVersion,
	)

	// 获取升级特定成员的任务
	upgradeTasks := e.componentMgr.GetMemberUpgradeTasks(
		manifest,
		memberIdx,
		status,
	)

	for _, task := range upgradeTasks {
		_, err := e.taskExecutor.Execute(ctx, task, nil)
		if err != nil {
			status.MemberStatuses[memberIdx] = "UpgradeFailed"
			return EtcdStateUpgradeMember, err
		}
	}

	status.MemberStatuses[memberIdx] = "UpgradeInProgress"
	return EtcdStateVerifyMember, nil
}

func (e *EnsureEtcdUpgradeRefactor) handleVerifyMember(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	memberIdx := status.CurrentMemberIdx
	
	// 检查成员是否已升级
	memberVersion, err := e.getEtcdMemberVersion(ctx, memberIdx)
	if err != nil {
		return EtcdStateVerifyMember, err
	}

	if memberVersion != status.TargetVersion {
		log.Info("Member not yet upgraded",
			"index", memberIdx,
			"version", memberVersion)
		return EtcdStateVerifyMember, nil
	}

	log.Info("Member upgraded successfully",
		"index", memberIdx,
		"version", memberVersion)

	status.MemberStatuses[memberIdx] = "Upgraded"
	status.CurrentMemberIdx++

	// 检查集群健康
	manifest, _ := e.componentMgr.GetComponentVersionManifest(
		ctx,
		"etcd",
		status.TargetVersion,
	)

	healthCheckTasks := e.componentMgr.GetHealthCheckTasks(manifest)

	for _, task := range healthCheckTasks {
		_, err := e.taskExecutor.Execute(ctx, task, nil)
		if err != nil {
			return EtcdStateVerifyMember, nil  // 重试
		}
	}

	// 如果所有成员都已升级，进入最终验证
	if status.CurrentMemberIdx >= status.TotalMembers {
		return EtcdStateAllUpgraded, nil
	}

	return EtcdStateUpgradeMember, nil
}

func (e *EnsureEtcdUpgradeRefactor) handleVerifyCluster(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Info("Verifying final etcd cluster state")

	manifest, _ := e.componentMgr.GetComponentVersionManifest(
		ctx,
		"etcd",
		status.TargetVersion,
	)

	verifyTasks := e.componentMgr.GetVerificationTasks(manifest)

	for _, task := range verifyTasks {
		_, err := e.taskExecutor.Execute(ctx, task, nil)
		if err != nil {
			return EtcdStateVerifyCluster, err
		}
	}

	log.Info("Etcd cluster verification passed")
	return EtcdStateCompleted, nil
}

func (e *EnsureEtcdUpgradeRefactor) handleRollback(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Error(nil, "Rolling back etcd upgrade", "reason", status.Error)

	manifest, _ := e.componentMgr.GetComponentVersionManifest(
		ctx,
		"etcd",
		status.SourceVersion,
	)

	rollbackTasks := e.componentMgr.GetRollbackTasks(manifest, status)

	for _, task := range rollbackTasks {
		_, err := e.taskExecutor.Execute(ctx, task, nil)
		if err != nil {
			log.Error(err, "Rollback task failed", "name", task.Name)
			return EtcdStateFailed, err
		}
	}

	log.Info("Etcd rolled back successfully")
	return EtcdStateFailed, fmt.Errorf("upgrade failed and rolled back: %s", status.Error)
}

	ctx context.Context,
	memberIdx int,
) (string, error) {
	// 查询特定 etcd 成员的版本
	// 实现省略
	return "", nil
```

### 1.1 核心理念：声明式组件生命周期管理
```
┌─────────────────────────────────────────────────────────────────────────┐
│                         设计核心原则                                    │
├─────────────────────────────────────────────────────────────────────────┤

│                                                                         │
```

```go
func (e *EnsureNodesEnv) NeedExecute(old, new *bkev1beta1.BKECluster) bool {
	// 如果没有旧版本，肯定需要初始化
	if old == nil {

	}

	newManifests := e.componentManager.GetNodeEnvManifests(new)

// Execute：核心执行逻辑
func (e *EnsureNodesEnv) Execute() (ctrl.Result, error) {
	ctx, _, bkeCluster, _, log := e.Ctx.Untie()
	
	// Step 1: 加载组件版本清单
	log.Info("Loading component manifests for NodeEnv", "version", bkeCluster.Spec.ClusterConfig.Cluster.KubernetesVersion)
	
	nodeEnvManifest, err := e.componentManager.GetComponentVersionManifest(ctx, 
		component.NodeEnvComponent, 
		bkeCluster.Spec.ClusterConfig.Cluster.KubernetesVersion,
	)
	if err != nil {
		return ctrl.Result{}, fmt.Errorf("load component manifest: %w", err)
	}
	
	// Step 2: 检查前置条件（兼容性、依赖等）
	if err := e.validatePrerequisites(ctx, bkeCluster, nodeEnvManifest); err != nil {
		return ctrl.Result{}, fmt.Errorf("prerequisite check failed: %w", err)
	}
	
	// Step 3: 比较版本并生成任务序列
	nodes, err := e.Ctx.NodeFetcher().GetBKENodesWrapperForCluster(ctx, bkeCluster)
	if err != nil {
		return ctrl.Result{}, fmt.Errorf("get nodes: %w", err)
	}
	
	tasks, err := e.componentManager.GenerateUpgradeTasks(
		bkeCluster,
		nodes,
		nodeEnvManifest,
	)
	if err != nil {
		return ctrl.Result{}, fmt.Errorf("generate upgrade tasks: %w", err)
	}
	
	// Step 4: 执行任务序列
	if err := e.executeTasks(ctx, tasks, bkeCluster, log); err != nil {
		return ctrl.Result{RequeueAfter: 10 * time.Second}, fmt.Errorf("execute tasks: %w", err)
	}
	
	// Step 5: 校验并更新状态
	if err := e.verifyAndUpdateStatus(ctx, bkeCluster, nodeEnvManifest, log); err != nil {
		return ctrl.Result{RequeueAfter: 10 * time.Second}, fmt.Errorf("verify and update status: %w", err)
	}
	
	log.Info("EnsureNodesEnv completed successfully")
	return ctrl.Result{}, nil
}

// 校验前置条件
func (e *EnsureNodesEnv) validatePrerequisites(
	ctx context.Context,
	cluster *bkev1beta1.BKECluster,
	manifest *component.LifecycleManifest,
) error {
	// 检查兼容性
	for _, dep := range manifest.ComponentVersion.Compatibility.Dependencies {
		otherManifest, err := e.componentManager.GetComponentVersionManifest(ctx,
			dep.Component,
			cluster.Spec.ClusterConfig.Cluster.KubernetesVersion,
		)
		if err != nil {
			return fmt.Errorf("dependency %s not found: %w", dep.Component, err)
		}
		
		if !isVersionInRange(otherManifest.ComponentVersion.Version, dep.MinVersion, dep.MaxVersion) {
			return fmt.Errorf("incompatible %s version: %s (requires %s-%s)",
				dep.Component, otherManifest.ComponentVersion.Version, dep.MinVersion, dep.MaxVersion)
		}
	}
	
	return nil
}

// 执行任务序列
func (e *EnsureNodesEnv) executeTasks(
	ctx context.Context,
	tasks []*component.Task,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) error {
	for _, task := range tasks {
		log.Info("Executing task", "name", task.Name, "type", task.Type)
		
		// 为每个节点执行任务
		taskCtx := &TaskExecutionContext{
			Cluster:          cluster,
			ComponentVersion: task.ComponentVersion,
			Nodes:            task.TargetNodes,
			Env:              task.Env,
		}
		
		// 调用通用任务执行器
		result, err := e.Ctx.TaskExecutor().Execute(ctx, task, taskCtx)
		if err != nil {
			if task.OnFailure == "Fail" {
				return fmt.Errorf("task %s failed: %w", task.Name, err)
			}
			log.Error(err, "Task failed but continuing", "name", task.Name)
		}
		
		// 等待任务完成（如果是异步的）
		if result.IsAsync {
			if err := e.waitForTaskCompletion(ctx, result.TaskID, task.Timeout); err != nil {
				return fmt.Errorf("wait for task completion: %w", err)
			}
		}
	}
	
	return nil
}
```

```go
package phases

import (
	"context"
	"fmt"
	"time"
	
	ctrl "sigs.k8s.io/controller-runtime"
	
	"gopkg.openfuyao.cn/cluster-api-provider-bke/pkg/component"
	"gopkg.openfuyao.cn/cluster-api-provider-bke/pkg/phaseframe"
)

type EnsureContainerRuntime struct {
	phaseframe.BasePhase
	componentManager *component.Manager
}

func NewEnsureContainerRuntime(ctx *phaseframe.PhaseContext) phaseframe.Phase {
	return &EnsureContainerRuntime{
		BasePhase:        phaseframe.NewBasePhase(ctx, "EnsureContainerRuntime"),
		componentManager: component.NewManager(ctx.Client),
	}
}

func (e *EnsureContainerRuntime) NeedExecute(old, new *bkev1beta1.BKECluster) bool {
	// 检查容器运行时版本是否变化
	if old == nil {
		return true
	}
	
	oldCR := old.Spec.ClusterConfig.Cluster.ContainerRuntime
	newCR := new.Spec.ClusterConfig.Cluster.ContainerRuntime
	
	// 版本变化 或 配置变化
	return oldCR.Runtime != newCR.Runtime ||
		oldCR.Version != newCR.Version ||
		!reflect.DeepEqual(oldCR.Config, newCR.Config)
}

func (e *EnsureContainerRuntime) Execute() (ctrl.Result, error) {
	ctx, _, cluster, _, log := e.Ctx.Untie()
	
	runtime := cluster.Spec.ClusterConfig.Cluster.ContainerRuntime
	
	// 加载组件清单
	manifest, err := e.componentManager.GetComponentVersionManifest(ctx,
		fmt.Sprintf("container-runtime-%s", runtime.Runtime),
		runtime.Version,
	)
	if err != nil {
		return ctrl.Result{}, fmt.Errorf("load manifest: %w", err)
	}
	
	// 判断是新安装还是升级
	nodes, _ := e.Ctx.NodeFetcher().GetBKENodesWrapperForCluster(ctx, cluster)
	
	var needUpgrade bool
	for _, node := range nodes.Items {
		if node.Status.ContainerRuntime.Version != runtime.Version {
			needUpgrade = true
			break
		}
	}
	
	if needUpgrade {
		return e.upgradeContainerRuntime(ctx, cluster, manifest, log)
	}
	
	return e.installContainerRuntime(ctx, cluster, manifest, log)
}

// 首次安装
func (e *EnsureContainerRuntime) installContainerRuntime(
	ctx context.Context,
	cluster *bkev1beta1.BKECluster,
	manifest *component.LifecycleManifest,
	log logr.Logger,
) (ctrl.Result, error) {
	log.Info("Installing container runtime", "type", manifest.ComponentVersion.Component, 
		"version", manifest.ComponentVersion.Version)
	
	// 执行 install manifest 中的所有步骤
	for _, step := range manifest.Lifecycle.Install.Steps {
		if err := e.executeStep(ctx, step, cluster, log); err != nil {
			if step.OnFailure == "Fail" {
				return ctrl.Result{}, err
			}
			log.Error(err, "Step failed but continuing", "step", step.Name)
		}
	}
	
	// 验证
	if err := e.verifyInstallation(ctx, cluster, manifest.Lifecycle.Verify, log); err != nil {
		return ctrl.Result{RequeueAfter: 10 * time.Second}, err
	}
	
	return ctrl.Result{}, nil
}

// 升级
func (e *EnsureContainerRuntime) upgradeContainerRuntime(
	ctx context.Context,
	cluster *bkev1beta1.BKECluster,
	newManifest *component.LifecycleManifest,
	log logr.Logger,
) (ctrl.Result, error) {
	log.Info("Upgrading container runtime", "target", newManifest.ComponentVersion.Version)
	
	runtime := cluster.Spec.ClusterConfig.Cluster.ContainerRuntime
	
	// 获取升级 manifest（版本特定）
	upgradeManifestName := fmt.Sprintf("upgrade-%s-to-%s", "1.7.1", runtime.Version) // 从当前状态读取
	upgradeManifest, err := e.componentManager.GetUpgradeManifest(ctx, upgradeManifestName)
	if err != nil {
		log.Info("No version-specific upgrade manifest found, using generic upgrade", "error", err)
		upgradeManifest = e.generateGenericUpgradeManifest(newManifest)
	}
	
	// 执行升级步骤
	for _, step := range upgradeManifest.Steps {
		if err := e.executeStep(ctx, step, cluster, log); err != nil {
			if upgradeManifest.Rollback.Enabled {
				log.Error(err, "Upgrade failed, rolling back")
				e.rollbackUpgrade(ctx, upgradeManifest.Rollback, log)
			}
```

## 四、有状态 Phase 整改方案：etcd 示例
### 4.1 Etcd 升级状态机
```go
package phases

import (
	"context"
	"fmt"
	"time"
	
	ctrl "sigs.k8s.io/controller-runtime"
	
	"gopkg.openfuyao.cn/cluster-api-provider-bke/pkg/component"
)

type EnsureEtcd struct {
	phaseframe.BasePhase
	componentManager *component.Manager
	stateManager     *EtcdStateManager
}

// Etcd 升级状态机
type EtcdUpgradeState string

const (
	EtcdStateIdle                EtcdUpgradeState = "Idle"
	EtcdStateBackupPreUpgrade    EtcdUpgradeState = "BackupPreUpgrade"
	EtcdStateCheckClusterHealth  EtcdUpgradeState = "CheckClusterHealth"
	EtcdStateUpgradingMember     EtcdUpgradeState = "UpgradingMember"
	EtcdStateVerifyMemberUpgrade EtcdUpgradeState = "VerifyMemberUpgrade"
	EtcdStateCheckClusterHealthAfterMember EtcdUpgradeState = "CheckClusterHealthAfterMember"
	EtcdStateAllMembersUpgraded  EtcdUpgradeState = "AllMembersUpgraded"
	EtcdStateVerifyFinalCluster  EtcdUpgradeState = "VerifyFinalCluster"
	EtcdStateUpgradeCompleted    EtcdUpgradeState = "UpgradeCompleted"
	EtcdStateRollingBack         EtcdUpgradeState = "RollingBack"
	EtcdStateFailed              EtcdUpgradeState = "Failed"
)

func (e *EnsureEtcd) Execute() (ctrl.Result, error) {
	ctx, _, cluster, _, log := e.Ctx.Untie()
	
	// 获取当前升级状态
	status := e.getEtcdUpgradeStatus(cluster)
	log.Info("etcd upgrade state", "state", status.State, "member", status.CurrentMember, "totalMembers", status.TotalMembers)
	
	// 状态机驱动
	nextState, err := e.handleState(ctx, status, cluster, log)
	if err != nil {
		// 失败，进入回滚状态
		if status.State != EtcdStateRollingBack && status.State != EtcdStateFailed {
			e.setEtcdUpgradeStatus(cluster, EtcdStateRollingBack, err.Error())
			return ctrl.Result{RequeueAfter: 5 * time.Second}, nil
		}
	}
	
	// 更新状态
	if nextState != status.State {
		e.setEtcdUpgradeStatus(cluster, nextState, "")
		return ctrl.Result{RequeueAfter: 10 * time.Second}, nil
	}
	
	return ctrl.Result{}, nil
}

func (e *EnsureEtcd) handleState(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	switch status.State {
	case EtcdStateIdle:
		return e.handleStateIdle(ctx, status, cluster, log)
	
	case EtcdStateBackupPreUpgrade:
		return e.handleStateBackupPreUpgrade(ctx, status, cluster, log)
	
	case EtcdStateCheckClusterHealth:
		return e.handleStateCheckClusterHealth(ctx, status, cluster, log)
	
	case EtcdStateUpgradingMember:
		return e.handleStateUpgradingMember(ctx, status, cluster, log)
	
	case EtcdStateVerifyMemberUpgrade:
		return e.handleStateVerifyMemberUpgrade(ctx, status, cluster, log)
	
	case EtcdStateCheckClusterHealthAfterMember:
		return e.handleStateCheckClusterHealthAfterMember(ctx, status, cluster, log)
	
	case EtcdStateAllMembersUpgraded:
		return e.handleStateAllMembersUpgraded(ctx, status, cluster, log)
	
	case EtcdStateVerifyFinalCluster:
		return e.handleStateVerifyFinalCluster(ctx, status, cluster, log)
	
	case EtcdStateUpgradeCompleted:
		return EtcdStateUpgradeCompleted, nil
	
	case EtcdStateRollingBack:
		return e.handleStateRollingBack(ctx, status, cluster, log)
	
	case EtcdStateFailed:
		return EtcdStateFailed, nil
	
	default:
		return "", fmt.Errorf("unknown state: %s", status.State)
	}
}

// State: Idle → 检查版本
func (e *EnsureEtcd) handleStateIdle(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Info("Checking if etcd upgrade is needed")
	
	etcdVersion := cluster.Spec.ClusterConfig.Cluster.EtcdVersion
	currentVersion := e.getCurrentEtcdVersion(cluster)
	
	if etcdVersion == currentVersion {
		log.Info("etcd already at desired version", "version", etcdVersion)
		return EtcdStateUpgradeCompleted, nil
	}
	
	log.Info("etcd upgrade needed", "from", currentVersion, "to", etcdVersion)
	
	// 初始化升级状态
	status.SourceVersion = currentVersion
	status.TargetVersion = etcdVersion
	status.BackupPath = fmt.Sprintf("/opt/etcd-backup/pre-upgrade-%s", time.Now().Format("20060102-150405"))
	status.CurrentMember = 0
	status.TotalMembers = 0
	
	return EtcdStateBackupPreUpgrade, nil
}

// State: BackupPreUpgrade → 快照备份
func (e *EnsureEtcd) handleStateBackupPreUpgrade(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Info("Creating etcd snapshot backup", "path", status.BackupPath)
	
	manifest, _ := e.componentManager.GetComponentVersionManifest(ctx, "etcd", status.TargetVersion)
	
	// 获取快照步骤
	backupStep := manifest.Lifecycle.Upgrade.GetStep("snapshot-backup")
	
	// 在主 etcd 节点执行备份
	task := &component.Task{
		Name: "backup-etcd",
		Type: backupStep.Type,
		Script: backupStep.Script,
		Env: map[string]string{
			"BACKUP_PATH": status.BackupPath,
		},
	}
	
	if result, err := e.Ctx.TaskExecutor().Execute(ctx, task, nil); err != nil {
		return EtcdStateBackupPreUpgrade, err
	}
	
	status.BackupPath = result.Output["backup_path"]
	log.Info("etcd snapshot created", "path", status.BackupPath)
	
	return EtcdStateCheckClusterHealth, nil
}

// State: CheckClusterHealth → 检查集群健康
func (e *EnsureEtcd) handleStateCheckClusterHealth(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Info("Checking etcd cluster health before upgrade")
	
	health, members, err := e.checkEtcdHealth(ctx, cluster)
	if err != nil {
		return EtcdStateCheckClusterHealth, fmt.Errorf("etcd health check failed: %w", err)
	}
	
	if !health.IsHealthy {
		return EtcdStateCheckClusterHealth, fmt.Errorf("etcd cluster not healthy: %v", health.UnhealthyMembers)
	}
	
	status.TotalMembers = len(members)
	status.MemberStatus = make(map[int]string, len(members))
	for i, member := range members {
		status.MemberStatus[i] = "Ready"
	}
	
	log.Info("etcd cluster is healthy", "members", len(members))
	
	return EtcdStateUpgradingMember, nil
}

// State: UpgradingMember → 升级单个成员
func (e *EnsureEtcd) handleStateUpgradingMember(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	if status.CurrentMember >= status.TotalMembers {
		return EtcdStateAllMembersUpgraded, nil
	}
	
	memberIndex := status.CurrentMember
	log.Info("Upgrading etcd member", "index", memberIndex, "total", status.TotalMembers)
	
	manifest, _ := e.componentManager.GetComponentVersionManifest(ctx, "etcd", status.TargetVersion)
	
	// 获取升级步骤
	upgradeMemberStep := manifest.Lifecycle.Upgrade.GetStep("rolling-upgrade")
	
	// 让该成员让出 leader
	e.drainEtcdLeadership(ctx, memberIndex, log)
	
	// 停止 etcd 进程
	// 备份数据（已在全局备份中完成）
	// 升级二进制
	// 启动新版本
	
	task := &component.Task{
		Name: "upgrade-etcd-member",
		Type: upgradeMemberStep.Type,
		Params: upgradeMemberStep.Params,
		Env: map[string]string{
			"MEMBER_INDEX":      fmt.Sprintf("%d", memberIndex),
			"SOURCE_VERSION":    status.SourceVersion,
			"TARGET_VERSION":    status.TargetVersion,
		},
	}
	
	if _, err := e.Ctx.TaskExecutor().Execute(ctx, task, nil); err != nil {
		log.Error(err, "Failed to upgrade etcd member", "index", memberIndex)
		status.MemberStatus[memberIndex] = "UpgradeFailed"
		return EtcdStateUpgradingMember, err
	}
	
	status.MemberStatus[memberIndex] = "UpgradeInProgress"
	
	return EtcdStateVerifyMemberUpgrade, nil
}

// State: VerifyMemberUpgrade → 验证成员升级
func (e *EnsureEtcd) handleStateVerifyMemberUpgrade(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	memberIndex := status.CurrentMember
	log.Info("Verifying etcd member upgrade", "index", memberIndex)
	
	// 检查成员是否已升级
	memberVersion, err := e.getEtcdMemberVersion(ctx, cluster, memberIndex)
	if err != nil {
		return EtcdStateVerifyMemberUpgrade, err
	}
	
	if memberVersion != status.TargetVersion {
		log.Info("Member not yet upgraded", "index", memberIndex, "version", memberVersion)
		return EtcdStateVerifyMemberUpgrade, nil  // 重试
	}
	
	log.Info("Member upgraded successfully", "index", memberIndex, "version", memberVersion)
	status.MemberStatus[memberIndex] = "Upgraded"
	status.CurrentMember++
	
	return EtcdStateCheckClusterHealthAfterMember, nil
}

// State: CheckClusterHealthAfterMember → 升级后检查集群
func (e *EnsureEtcd) handleStateCheckClusterHealthAfterMember(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Info("Checking etcd cluster health after member upgrade", "member", status.CurrentMember-1)
	
	health, _, err := e.checkEtcdHealth(ctx, cluster)
	if err != nil {
		log.Error(err, "Health check failed, retrying")
		return EtcdStateCheckClusterHealthAfterMember, nil  // 重试
	}
	
	if !health.IsHealthy {
		return EtcdStateCheckClusterHealthAfterMember, fmt.Errorf("etcd cluster unhealthy after member upgrade")
	}
	
	log.Info("etcd cluster is healthy after member upgrade")
	
	// 检查是否所有成员都已升级
	if status.CurrentMember >= status.TotalMembers {
		return EtcdStateAllMembersUpgraded, nil
	}
	
	return EtcdStateUpgradingMember, nil
}

// State: AllMembersUpgraded → 所有成员已升级
func (e *EnsureEtcd) handleStateAllMembersUpgraded(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Info("All etcd members upgraded successfully")
	
	return EtcdStateVerifyFinalCluster, nil
}

// State: VerifyFinalCluster → 最终集群验证
func (e *EnsureEtcd) handleStateVerifyFinalCluster(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Info("Verifying final etcd cluster state")
	
	manifest, _ := e.componentManager.GetComponentVersionManifest(ctx, "etcd", status.TargetVersion)
	
	// 执行最终验证步骤
	verifyStep := manifest.Lifecycle.Upgrade.GetStep("post-upgrade-verify")
	
	task := &component.Task{
		Name: "verify-etcd-upgrade",
		Type: verifyStep.Type,
		Script: verifyStep.Script,
	}
	
	if _, err := e.Ctx.TaskExecutor().Execute(ctx, task, nil); err != nil {
		log.Error(err, "Final verification failed")
		return EtcdStateVerifyFinalCluster, err
	}
	
	log.Info("etcd cluster verification passed")
	
	return EtcdStateUpgradeCompleted, nil
}

// State: RollingBack → 回滚升级
func (e *EnsureEtcd) handleStateRollingBack(
	ctx context.Context,
	status *EtcdUpgradeStatus,
	cluster *bkev1beta1.BKECluster,
	log logr.Logger,
) (EtcdUpgradeState, error) {
	log.Error(nil, "Rolling back etcd upgrade", "reason", status.Error)
	
	// 从快照恢复
	if err := e.restoreEtcdFromSnapshot(ctx, status.BackupPath, log); err != nil {
		log.Error(err, "Snapshot restore failed")
		return EtcdStateFailed, err
	}
	
	log.Info("etcd restored from snapshot")
	
	return EtcdStateFailed, fmt.Errorf("upgrade failed and rolled back: %s", status.Error)
}
```

## 六、完整设计思路总结
### 6.1 分层模型
```
┌─────────────────────────────────────────────────────────────────────────┐
│                         Application Layer                                 │
│  BKECluster Controller → Phase Executor → Task Orchestrator              │
├─────────────────────────────────────────────────────────────────────────┤
│                      Component Layer                                       │
│  ComponentVersion Manifest ← → Component Manager ← → Compatibility Check  │
├─────────────────────────────────────────────────────────────────────────┤
│                        Execution Layer                                     │
│  Task Executor → {Builtin Plugins, Shell Scripts, K8s Resources}         │
├─────────────────────────────────────────────────────────────────────────┤
│                      Infrastructure Layer                                  │
│  BKE Agent ← → Nodes (OS, Binaries, Services)                           │
└─────────────────────────────────────────────────────────────────────────┘
```
### 6.2 核心原则
1. **声明式优于命令式**
   - 所有安装/卸载/升级逻辑声明在 YAML 中
   - 控制器只负责驱动状态机
2. **版本管理最优先**
   - 每个组件独立版本化
   - 版本对比 → 生成差异任务
3. **容错与回滚内置**
   - 每一步都有失败处理规则
   - 支持自动回滚
4. **状态机驱动有状态升级**
   - etcd 等有状态组件用状态机
   - 幂等操作，可重入
5. **兼容性前置校验**
   - 升级前检查依赖版本
   - 阻止不兼容升级

```go
// 生成迁移清单
func GenerateMigrationManifest(
	ctx context.Context,
	legacyVersions map[string]string,
	targetVersions map[string]string,
) (*component.MigrationManifest, error) {
	manifest := &component.MigrationManifest{
		Steps: []*component.MigrationStep{},
	}
	
	// 逐个组件生成迁移步骤
	for component, legacyVersion := range legacyVersions {
		targetVersion := targetVersions[component]
		
		if legacyVersion != targetVersion {
			// 需要升级
			manifest.Steps = append(manifest.Steps, &component.MigrationStep{
				Type:       "upgrade",
				Component:  component,
				From:       legacyVersion,
				To:         targetVersion,
				Order:      1,  // 执行顺序
			})
		} else {
			// 仅需导入配置
			manifest.Steps = append(manifest.Steps, &component.MigrationStep{
				Type:       "import",
				Component:  component,
				Version:    legacyVersion,
				Order:      1,
			})
		}
	}
	
	return manifest, nil
}

// 执行迁移
func ExecuteMigration(
	ctx context.Context,
	manifest *component.MigrationManifest,
) error {
	for _, step := range manifest.Steps {
		fmt.Printf("Executing migration step: %s %s from %s to %s\n",
			step.Type, step.Component, step.From, step.To)
		
		switch step.Type {
		case "upgrade":
			// 调用升级逻辑
			if err := executeUpgrade(ctx, step); err != nil {
				return err
			}
		
		case "import":
			// 导入配置
			if err := importConfig(ctx, step); err != nil {
				return err
			}
		}
	}
	
	return nil
}
```

## 总结：完整设计要点

| 设计要点 | 说明 |
|---------|------|
| **声明式优于��令式** | 所有 LCM 逻辑在 Manifest 中，控制器只调和 |
| **版本即一等公民** | ComponentVersion CRD 管理所有版本信息 |
| **兼容性前置** | 升级前检查依赖版本，阻止不兼容操作 |
| **有状态支持** | 状态机驱动 etcd 等有状态组件升级 |
| **容错与回滚** | 每一步都有失败处理和自动回滚 |
| **幂等操作** | 所有步骤可重入，支持故障恢复 |
| **模块化任务执行** | shell、builtin、k8s 三种执行方式 |
| **渐进式迁移** | 旧系统自动检测与导入，无缝升级 |
| **可观测性** | 完整的状态机日志和事件 |
| **工作量 16 周** | 2-3 人团队完成整改 |

这个方案通过分层架构、声明式定义、版本管理、有状态支持和完整容错机制，实现了 BKE 从命令式到声明式、从手工运维到自动化升级的转变，大大提升了集群管理的可靠性和可维护性。
