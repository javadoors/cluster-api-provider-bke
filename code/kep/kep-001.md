## 1. 摘要
本提案针对 `cluster-api-provider-bke` 项目进行架构级重构，解决当前存在的五大核心缺陷：不符合 Cluster API 标准架构、Phase Flow Engine 复杂度过高、升级机制不完善、多 OS 支持硬编码、缺少 UPI 场景支持。重构目标是使项目遵循 Cluster API Provider 规范，支持三层集群架构（引导/管理/业务集群），集成声明式升级能力，并提高代码的可维护性和可扩展性。
## 2. 动机
### 2.1 当前架构问题
当前 `cluster-api-provider-bke` 存在以下关键缺陷：

**缺陷1：不符合 Cluster API 标准架构**

当前实现将 Bootstrap、Control Plane 和 Infrastructure 逻辑全部内置在 Infrastructure Provider 中，违反了 Cluster API 的 Provider 分离原则：
```
当前架构（不符合规范）：
┌──────────────────────────────────────┐
│        BKEClusterReconciler          │
│  ├── 基础设施管理                    │
│  ├── 引导配置（kubeadm init/join）   │
│  ├── 控制平面管理                    │
│  ├── 证书管理                        │
│  ├── 负载均衡管理                    │
│  └── Addon 部署                      │
└──────────────────────────────────────┘

标准 Cluster API 架构：
┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐
│ Infrastructure  │ │   Bootstrap     │ │  ControlPlane   │
│    Provider     │ │    Provider     │ │    Provider     │
│  • BKECluster   │ │  • KubeadmConfig│ │  • KCP          │
│  • BKEMachine   │ │  • Ignition     │ │  • BKEControlPl │
└─────────────────┘ └─────────────────┘ └─────────────────┘
```

**缺陷2：Phase Flow Engine 复杂度过高**
- 使用 `panic/recover` 处理错误不规范
- goroutine 生命周期管理不清晰（`WatchBKEClusterStatus`）
- 阶段依赖关系隐式定义
- 阶段状态管理复杂，难以追踪和调试

**缺陷3：升级机制不完善**
- 仅检查 `OpenFuyaoVersion` 变化，判断维度单一
- 缺少版本兼容性检查和回滚机制
- 脚本式升级，无声明式状态管理
- 缺少升级速率和并发数控制

**缺陷4：多 OS 支持硬编码**
- 新增 OS 需修改代码
- 缺少 OS 特性抽象层

**缺陷5：缺少 UPI 场景支持**
- 用户无法使用已有基础设施（LB、DNS、节点）
- 缺少用户自提供资源的验证机制
### 2.2 与业界标准的差距
| 维度 | OpenShift Installer | 当前 BKE | 差距 |
|------|---------------------|----------|------|
| 安装模式 | IPI/UPI 统一架构 | 仅 IPI 模式 | 缺少 UPI 支持 |
| 配置管理 | Asset 依赖图 | 直接 CR 创建 | 缺少状态追踪 |
| 升级机制 | CVO 声明式升级 | 脚本式升级 | 缺少回滚能力 |
| OS 支持 | MachineConfig 抽象 | 硬编码适配 | 扩展性差 |
| 节点配置 | Ignition 声明式 | SSH + Command | 安全性不足 |
## 3. 目标
1. **遵循 Cluster API 规范**：实现标准的 Infrastructure/Bootstrap/ControlPlane Provider 分离
2. **支持三层集群架构**：引导集群 → 管理集群 → 业务集群
3. **声明式升级能力**：引入 ClusterVersion CRD + CVO 控制器
4. **UPI/IPI 双模式**：引入 InfrastructureMode 支持
5. **可扩展 OS 支持**：引入 OSProvider 接口 + 注册机制
6. **状态机驱动**：使用状态机模式重构 Phase Flow Engine

### 4.2 状态机驱动的 Phase Flow Engine
使用状态机模式替代当前的 Phase Flow 实现：
```go
// pkg/statemachine/state.go
type ClusterPhase string

const (
    PhasePending      ClusterPhase = "Pending"
    PhaseProvisioning ClusterPhase = "Provisioning"
    PhaseRunning      ClusterPhase = "Running"
    PhaseUpdating     ClusterPhase = "Updating"
    PhaseDeleting     ClusterPhase = "Deleting"
    PhaseFailed       ClusterPhase = "Failed"
)

type State interface {
    Name() ClusterPhase
    Enter(ctx context.Context) error
    Execute(ctx context.Context) (ClusterPhase, error)
    Exit(ctx context.Context) error
    CanTransitionTo(target ClusterPhase) bool
}

type ClusterStateMachine struct {
    client       client.Client
    cluster      *bkev1beta1.BKECluster
    states       map[ClusterPhase]State
    currentState ClusterPhase
}

func (sm *ClusterStateMachine) Reconcile(ctx context.Context) (ctrl.Result, error) {
    state := sm.states[sm.currentState]
    nextPhase, err := state.Execute(ctx)
    if err != nil {
        sm.transitionTo(PhaseFailed)
        return ctrl.Result{}, err
    }
    if nextPhase != sm.currentState {
        if err := sm.transition(ctx, nextPhase); err != nil {
            return ctrl.Result{}, err
        }
    }
    return sm.getResult()
}
```
**状态转换矩阵**：
```
┌─────────────┬───────────────────────────────────────────────┐
│ 当前状态    │ 可转换目标状态                                │
├─────────────┼───────────────────────────────────────────────┤
│ Pending     │ → Provisioning, Deleting                      │
│ Provisioning│ → Running, Failed                             │
│ Running     │ → Updating, Deleting                          │
│ Updating    │ → Running, Failed, RollingBack                │
│ Deleting    │ → (terminated)                                │
│ Failed      │ → Pending (retry), Updating (retry)           │
└─────────────┴───────────────────────────────────────────────┘
```

### 4.4 统一错误处理
```go
// pkg/errors/errors.go
type ReconcileError struct {
    Type       ErrorType
    Message    string
    Cause      error
    RetryAfter time.Duration
}

type ErrorType string

const (
    ErrorTypeTransient  ErrorType = "Transient"
    ErrorTypePermanent  ErrorType = "Permanent"
    ErrorTypeConflict   ErrorType = "Conflict"
    ErrorTypeDependency ErrorType = "Dependency"
)

func (r *BKEClusterReconciler) handleError(ctx context.Context, cluster *bkev1beta1.BKECluster, err error) (ctrl.Result, error) {
    var reconcileErr *ReconcileError
    if errors.As(err, &reconcileErr) {
        switch reconcileErr.Type {
        case ErrorTypeTransient:
            return ctrl.Result{RequeueAfter: 5 * time.Second}, nil
        case ErrorTypePermanent:
            conditions.Set(cluster, &clusterv1.Condition{
                Type:    clusterv1.ReadyCondition,
                Status:  corev1.ConditionFalse,
                Reason:  "PermanentError",
                Message: err.Error(),
            })
            return ctrl.Result{}, nil
        case ErrorTypeDependency:
            return ctrl.Result{RequeueAfter: reconcileErr.RetryAfter}, nil
        }
    }
    return ctrl.Result{}, err
}
```

### 5.2 兼容性保证
为确保平滑迁移，实现兼容性层：
```go
// pkg/compat/compat.go
type CompatibilityLayer struct {
    client client.Client
}

func (c *CompatibilityLayer) ConvertToNewResources(
    ctx context.Context,
    bkeCluster *bkev1beta1.BKECluster,
    cluster *clusterv1.Cluster,
) error {
    controlPlane := c.createControlPlane(bkeCluster, cluster)
    if err := c.client.Create(ctx, controlPlane); err != nil {
        return err
    }
    configTemplate := c.createConfigTemplate(bkeCluster, cluster)
    if err := c.client.Create(ctx, configTemplate); err != nil {
        return err
    }
    cluster.Spec.ControlPlaneRef = &clusterv1.ContractVersionedObjectReference{
        APIVersion: controlplanev1.GroupVersion.String(),
        Kind:       "BKEControlPlane",
        Name:       controlPlane.Name,
    }
    return c.client.Update(ctx, cluster)
}
```

## 8. 验收标准
1. BKECluster、BKEKubeadmConfig、BKEControlPlane 三个 Provider 独立运行
2. 支持 IPI 和 UPI 两种安装模式
3. 升级流程支持 PreCheck → Upgrade → PostCheck → Rollback 完整链路
4. 新增 OS 仅需实现 OSProvider 接口，无需修改核心代码
5. 所有现有功能通过集成测试验证
6. 状态机驱动的 Phase 执行可追踪、可恢复

```go
package upi

import (
    "context"
    "fmt"
    "net"
    "time"
    
    "k8s.io/apimachinery/pkg/util/validation/field"
    bkev1beta1 "gopkg.openfuyao.cn/cluster-api-provider-bke/api/capbke/v1beta1"
)

// Validator UPI 验证器
type Validator struct {
    sshClient SSHClient
    dnsClient DNSClient
}

// NewValidator 创建验证器
func NewValidator(sshClient SSHClient, dnsClient DNSClient) *Validator {
    return &Validator{
        sshClient: sshClient,
        dnsClient: dnsClient,
    }
}

// Validate 验证用户提供的资源
func (v *Validator) Validate(ctx context.Context, bkeCluster *bkev1beta1.BKECluster) field.ErrorList {
    var allErrs field.ErrorList
    
    if bkeCluster.Spec.InfrastructureMode != bkev1beta1.InfrastructureModeUPI {
        return allErrs
    }
    
    upi := bkeCluster.Spec.UserProvidedInfrastructure
    if upi == nil {
        allErrs = append(allErrs, field.Required(
            field.NewPath("spec", "userProvidedInfrastructure"),
            "required for UPI mode",
        ))
        return allErrs
    }
    
    // 验证负载均衡器
    if upi.LoadBalancer != nil {
        if errs := v.validateLoadBalancer(ctx, upi.LoadBalancer); len(errs) > 0 {
            allErrs = append(allErrs, errs...)
        }
    }
    
    // 验证 DNS
    if upi.DNS != nil {
        if errs := v.validateDNS(ctx, upi.DNS); len(errs) > 0 {
            allErrs = append(allErrs, errs...)
        }
    }
    
    // 验证节点
    if errs := v.validateNodes(ctx, upi.Nodes); len(errs) > 0 {
        allErrs = append(allErrs, errs...)
    }
    
    return allErrs
}

// validateLoadBalancer 验证负载均衡器
func (v *Validator) validateLoadBalancer(ctx context.Context, lb *bkev1beta1.LoadBalancerRef) field.ErrorList {
    var errs field.ErrorList
    path := field.NewPath("spec", "userProvidedInfrastructure", "loadBalancer")
    
    // 验证端点可达性
    if err := v.checkEndpointReachability(ctx, lb.Endpoint, lb.Port); err != nil {
        errs = append(errs, field.Invalid(path.Child("endpoint"), lb.Endpoint, 
            fmt.Sprintf("endpoint not reachable: %v", err)))
    }
    
    // 验证健康检查
    if lb.HealthCheckPath != "" {
        if err := v.checkHealthEndpoint(ctx, lb.Endpoint, lb.Port, lb.HealthCheckPath); err != nil {
            errs = append(errs, field.Invalid(path.Child("healthCheckPath"), lb.HealthCheckPath,
                fmt.Sprintf("health check failed: %v", err)))
        }
    }
    
    return errs
}

// validateDNS 验证 DNS 配置
func (v *Validator) validateDNS(ctx context.Context, dns *bkev1beta1.DNSRef) field.ErrorList {
    var errs field.ErrorList
    path := field.NewPath("spec", "userProvidedInfrastructure", "dns")
    
    // 验证 DNS 服务器可达性
    if err := v.dnsClient.CheckServer(ctx, dns.Server); err != nil {
        errs = append(errs, field.Invalid(path.Child("server"), dns.Server,
            fmt.Sprintf("DNS server not reachable: %v", err)))
    }
    
    return errs
}

// validateNodes 验证节点列表
func (v *Validator) validateNodes(ctx context.Context, nodes []bkev1beta1.NodeRef) field.ErrorList {
    var errs field.ErrorList
    path := field.NewPath("spec", "userProvidedInfrastructure", "nodes")
    
    if len(nodes) == 0 {
        errs = append(errs, field.Required(path, "at least one node is required"))
        return errs
    }
    
    for i, node := range nodes {
        nodePath := path.Index(i)
        
        // 验证 IP 格式
        if net.ParseIP(node.IP) == nil {
            errs = append(errs, field.Invalid(nodePath.Child("ip"), node.IP, "invalid IP address"))
        }
        
        // 验证 SSH 连接
        if err := v.sshClient.CheckConnection(ctx, node.IP, 22, 5*time.Second); err != nil {
            errs = append(errs, field.Invalid(nodePath.Child("ip"), node.IP,
                fmt.Sprintf("SSH connection failed: %v", err)))
        }
    }
    
    return errs
}

// checkEndpointReachability 检查端点可达性
func (v *Validator) checkEndpointReachability(ctx context.Context, host string, port int32) error {
    timeout := 5 * time.Second
    conn, err := net.DialTimeout("tcp", fmt.Sprintf("%s:%d", host, port), timeout)
    if err != nil {
        return err
    }
    conn.Close()
    return nil
}
```

```go
package capbke

import (
    "context"

    "gopkg.openfuyao.cn/cluster-api-provider-bke/pkg/upi"
)

// BKEClusterReconciler 重构后的控制器
type BKEClusterReconciler struct {
    client.Client
    Scheme   *runtime.Scheme
    Recorder record.EventRecorder
    
    // 分离的子组件
    validator    *ClusterValidator
    provisioner  *ClusterProvisioner
    monitor      *ClusterMonitor
    statusManager *StatusManager
    
    // UPI 验证器
    upiValidator *upi.Validator
}

// Reconcile 主协调循环
func (r *BKEClusterReconciler) Reconcile(ctx context.Context, req ctrl.Request) (ctrl.Result, error) {
    // 1. 获取集群资源
    cluster, err := r.getCluster(ctx, req)
    if err != nil {
        return ctrl.Result{}, client.IgnoreNotFound(err)
    }
    
    // 2. 验证配置
    if err := r.validator.Validate(ctx, cluster); err != nil {
        return r.handleError(ctx, cluster, err)
    }
    
    // 3. UPI 模式额外验证
    if cluster.Spec.InfrastructureMode == bkev1beta1.InfrastructureModeUPI {
        if errs := r.upiValidator.Validate(ctx, cluster); len(errs) > 0 {
            return r.handleError(ctx, cluster, errs.ToAggregate())
        }
    }
    
    // 4. 根据状态分发处理
    switch cluster.Status.Phase {
    case "":
        return r.provisioner.Initialize(ctx, cluster)
    case "Provisioning":
        return r.provisioner.Provision(ctx, cluster)
    case "Running":
        return r.monitor.HealthCheck(ctx, cluster)
    case "Deleting":
        return r.provisioner.Delete(ctx, cluster)
    default:
        return ctrl.Result{}, nil
    }
}
```

```go
import (
```

```go
    clusterv1 "sigs.k8s.io/cluster-api/api/v1beta1"

// BKEKubeadmConfigReconciler Bootstrap 配置控制器
type BKEKubeadmConfigReconciler struct {
    client.Client
    Scheme   *runtime.Scheme
    Recorder record.EventRecorder
    
    // 子组件
    dataGenerator *BootstrapDataGenerator
    osProvider    osprovider.Provider
}

// Reconcile 主协调循环
func (r *BKEKubeadmConfigReconciler) Reconcile(ctx context.Context, req ctrl.Request) (ctrl.Result, error) {
    log := ctrl.LoggerFrom(ctx)
    
    // 1. 获取 BKEKubeadmConfig
    config := &bootstrapv1.BKEKubeadmConfig{}
    if err := r.Get(ctx, req.NamespacedName, config); err != nil {
        return ctrl.Result{}, client.IgnoreNotFound(err)
    }
    
    // 2. 检查是否已就绪
    if config.Status.Ready {
        return ctrl.Result{}, nil
    }
    
    // 3. 获取关联的 Machine
    machine, err := r.getOwnerMachine(ctx, config)
    if err != nil {
        log.Error(err, "failed to get owner machine")
        return ctrl.Result{}, err
    }
    
    if machine == nil {
        log.Info("waiting for machine owner reference")
        return ctrl.Result{}, nil
    }
    
    // 4. 获取关联的 Cluster
    cluster, err := r.getClusterFromMachine(ctx, machine)
    if err != nil {
        log.Error(err, "failed to get cluster")
        return ctrl.Result{}, err
    }
    
    // 5. 生成引导数据
    bootstrapData, err := r.dataGenerator.Generate(ctx, config, machine, cluster)
    if err != nil {
        log.Error(err, "failed to generate bootstrap data")
        r.Recorder.Eventf(config, corev1.EventTypeWarning, "GenerateFailed", 
            "Failed to generate bootstrap data: %v", err)
        return ctrl.Result{}, err
    }
    
    // 6. 创建 Secret 存储引导数据
    secret := &corev1.Secret{
        ObjectMeta: metav1.ObjectMeta{
            Name:      fmt.Sprintf("%s-bootstrap-data", config.Name),
            Namespace: config.Namespace,
            OwnerReferences: []metav1.OwnerReference{
                {
                    APIVersion: bootstrapv1.GroupVersion.String(),
                    Kind:       "BKEKubeadmConfig",
                    Name:       config.Name,
                    UID:        config.UID,
                    Controller: pointer.Bool(true),
                },
            },
        },
        Data: map[string][]byte{
            "value": []byte(bootstrapData),
        },
    }
    
    if err := r.Create(ctx, secret); err != nil {
        if !errors.IsAlreadyExists(err) {
            log.Error(err, "failed to create bootstrap secret")
            return ctrl.Result{}, err
        }
    }
    
    // 7. 更新状态
    config.Status.Ready = true
    config.Status.DataSecretName = pointer.String(secret.Name)
    config.Status.Conditions = []metav1.Condition{
        {
            Type:               "Ready",
            Status:             metav1.ConditionTrue,
            Reason:             "DataGenerated",
            Message:            "Bootstrap data generated successfully",
            LastTransitionTime: metav1.Now(),
        },
    }
    
    if err := r.Status().Update(ctx, config); err != nil {
        log.Error(err, "failed to update status")
        return ctrl.Result{}, err
    }
    
    r.Recorder.Event(config, corev1.EventTypeNormal, "Ready", "Bootstrap data generated")
    return ctrl.Result{}, nil
}

// SetupWithManager 设置控制器
func (r *BKEKubeadmConfigReconciler) SetupWithManager(mgr ctrl.Manager) error {
    return ctrl.NewControllerManagedBy(mgr).
        For(&bootstrapv1.BKEKubeadmConfig{}).
        Watches(
            &source.Kind{Type: &clusterv1.Machine{}},
            handler.EnqueueRequestsFromMapFunc(r.machineToBKEKubeadmConfig),
        ).
        Complete(r)
}

// machineToBKEKubeadmConfig Machine 到 BKEKubeadmConfig 的映射
func (r *BKEKubeadmConfigReconciler) machineToBKEKubeadmConfig(o client.Object) []reconcile.Request {
    machine := o.(*clusterv1.Machine)
    
    if machine.Spec.Bootstrap.ConfigRef == nil {
        return nil
    }
    
    if machine.Spec.Bootstrap.ConfigRef.Kind != "BKEKubeadmConfig" {
        return nil
    }
    
    return []reconcile.Request{
        {
            NamespacedName: types.NamespacedName{
                Name:      machine.Spec.Bootstrap.ConfigRef.Name,
                Namespace: machine.Namespace,
            },
        },
    }
}
```

```go
    nodeManager     *ControlPlaneNodeManager
    certManager     *CertificateManager
    loadBalancerMgr *LoadBalancerManager
    etcdManager     *EtcdManager

// Reconcile 主协调循环
func (r *BKEControlPlaneReconciler) Reconcile(ctx context.Context, req ctrl.Request) (ctrl.Result, error) {
    log := ctrl.LoggerFrom(ctx)
    
    // 1. 获取 ControlPlane 资源
    controlPlane := &controlplanev1.BKEControlPlane{}
    if err := r.Get(ctx, req.NamespacedName, controlPlane); err != nil {
        return ctrl.Result{}, client.IgnoreNotFound(err)
    }
    
    // 2. 获取关联的 Cluster
    cluster, err := r.getOwnerCluster(ctx, controlPlane)
    if err != nil {
        log.Error(err, "failed to get owner cluster")
        return ctrl.Result{}, err
    }
    
    if cluster == nil {
        log.Info("waiting for cluster owner reference")
        return ctrl.Result{}, nil
    }
    
    // 3. 检查 Cluster 是否已删除
    if !cluster.DeletionTimestamp.IsZero() {
        return r.reconcileDelete(ctx, controlPlane, cluster)
    }
    
    // 4. 协调证书
    if err := r.certManager.Reconcile(ctx, controlPlane, cluster); err != nil {
        log.Error(err, "failed to reconcile certificates")
        return ctrl.Result{}, err
    }
    
    // 5. 协调负载均衡
    if err := r.loadBalancerMgr.Reconcile(ctx, controlPlane, cluster); err != nil {
        log.Error(err, "failed to reconcile load balancer")
        return ctrl.Result{}, err
    }
    
    // 6. 协调 Etcd
    if err := r.etcdManager.Reconcile(ctx, controlPlane, cluster); err != nil {
        log.Error(err, "failed to reconcile etcd")
        return ctrl.Result{}, err
    }
    
    // 7. 协调控制平面节点
    result, err := r.nodeManager.Reconcile(ctx, controlPlane, cluster)
    if err != nil {
        log.Error(err, "failed to reconcile control plane nodes")
        return ctrl.Result{}, err
    }
    
    // 8. 更新状态
    if err := r.updateStatus(ctx, controlPlane); err != nil {
        log.Error(err, "failed to update status")
        return ctrl.Result{}, err
    }
    
    return result, nil
}

    if err := r.certManager.Delete(ctx, controlPlane, cluster); err != nil {
        log.Error(err, "failed to delete certificates")
    }
```

```go
import (
    "context"
    "fmt"
    "sync"
    
    "github.com/go-logr/logr"

    "sigs.k8s.io/controller-runtime/pkg/client"

// ClusterStateMachine 集群状态机
type ClusterStateMachine struct {
    client       client.Client
    cluster      *bkev1beta1.BKECluster
    states       map[ClusterPhase]State
    currentState ClusterPhase
    log          logr.Logger
    mu           sync.RWMutex
}

// NewClusterStateMachine 创建集群状态机
func NewClusterStateMachine(client client.Client, cluster *bkev1beta1.BKECluster, log logr.Logger) *ClusterStateMachine {
    sm := &ClusterStateMachine{
        client:  client,
        cluster: cluster,
        states:  make(map[ClusterPhase]State),
        log:     log,
    }
    
    // 注册状态
    sm.registerStates()
    
    // 初始化当前状态
    sm.currentState = sm.getInitialPhase(cluster)
    
    return sm
}

// registerStates 注册所有状态
func (sm *ClusterStateMachine) registerStates() {
    sm.states[PhasePending] = NewPendingState(sm.client, sm.log)
    sm.states[PhaseProvisioning] = NewProvisioningState(sm.client, sm.log)
    sm.states[PhaseRunning] = NewRunningState(sm.client, sm.log)
    sm.states[PhaseUpdating] = NewUpdatingState(sm.client, sm.log)
    sm.states[PhaseDeleting] = NewDeletingState(sm.client, sm.log)
    sm.states[PhaseFailed] = NewFailedState(sm.client, sm.log)
}

// getInitialPhase 获取初始阶段
func (sm *ClusterStateMachine) getInitialPhase(cluster *bkev1beta1.BKECluster) ClusterPhase {
    if cluster.Status.Phase == "" {
        return PhasePending
    }
    return ClusterPhase(cluster.Status.Phase)
}

// CurrentPhase 返回当前阶段
func (sm *ClusterStateMachine) CurrentPhase() ClusterPhase {
    sm.mu.RLock()
    defer sm.mu.RUnlock()
    return sm.currentState
}

// Reconcile 执行协调
func (sm *ClusterStateMachine) Reconcile(ctx context.Context) (ctrl.Result, error) {
    sm.mu.Lock()
    defer sm.mu.Unlock()
    
    state, ok := sm.states[sm.currentState]
    if !ok {
        return ctrl.Result{}, fmt.Errorf("unknown state: %s", sm.currentState)
    }
    
    // 执行当前状态
    nextPhase, err := state.Execute(ctx)
    if err != nil {
        sm.log.Error(err, "state execution failed", "state", sm.currentState)
        sm.transitionTo(PhaseFailed)
        return ctrl.Result{}, err
    }
    
    // 状态转换
    if nextPhase != sm.currentState {
        if err := sm.transition(ctx, nextPhase); err != nil {
            return ctrl.Result{}, err
        }
    }
    
    return sm.getResult()
}

// transition 执行状态转换
func (sm *ClusterStateMachine) transition(ctx context.Context, targetPhase ClusterPhase) error {
    currentState := sm.states[sm.currentState]
    targetState := sm.states[targetPhase]
    
    if targetState == nil {
        return fmt.Errorf("target state not found: %s", targetPhase)
    }
    
    // 验证转换是否合法
    if !currentState.CanTransitionTo(targetPhase) {
        return fmt.Errorf("invalid transition from %s to %s", sm.currentState, targetPhase)
    }
    
    sm.log.Info("transitioning state", "from", sm.currentState, "to", targetPhase)
    
    // 退出当前状态
    if err := currentState.Exit(ctx); err != nil {
        return fmt.Errorf("exit state %s failed: %w", sm.currentState, err)
    }
    
    // 进入新状态
    if err := targetState.Enter(ctx); err != nil {
        return fmt.Errorf("enter state %s failed: %w", targetPhase, err)
    }
    
    // 更新状态
    sm.currentState = targetPhase
    sm.cluster.Status.Phase = string(targetPhase)
    
    // 持久化状态
    if err := sm.client.Status().Update(ctx, sm.cluster); err != nil {
        return fmt.Errorf("failed to update cluster status: %w", err)
    }
    
    return nil
}

// transitionTo 内部转换方法
func (sm *ClusterStateMachine) transitionTo(phase ClusterPhase) {
    sm.currentState = phase
    sm.cluster.Status.Phase = string(phase)
}

// getResult 获取返回结果
func (sm *ClusterStateMachine) getResult() (ctrl.Result, error) {
    switch sm.currentState {
    case PhaseProvisioning, PhaseUpdating:
        return ctrl.Result{RequeueAfter: 5 * time.Second}, nil
    case PhaseRunning:
        return ctrl.Result{RequeueAfter: 30 * time.Second}, nil
    case PhaseFailed:
        return ctrl.Result{RequeueAfter: 30 * time.Second}, nil
    default:
        return ctrl.Result{}, nil
    }
}
```

```go
type UpgradeCoordinator struct {
    client.Client
    
    detector *UpgradeDetector
    checker  *UpgradeChecker
    executor *UpgradeExecutor
    rollback *RollbackManager
}

// Coordinate 协调升级
func (c *UpgradeCoordinator) Coordinate(ctx context.Context, clusterVersion *upgradev1.ClusterVersion) error {
    switch clusterVersion.Status.Phase {
    case upgradev1.UpgradePhaseIdle:
        return c.handleIdle(ctx, clusterVersion)
    case upgradev1.UpgradePhasePreCheck:
        return c.handlePreCheck(ctx, clusterVersion)
    case upgradev1.UpgradePhasePreparing:
        return c.handlePreparing(ctx, clusterVersion)
    case upgradev1.UpgradePhaseUpgrading:
        return c.handleUpgrading(ctx, clusterVersion)
    case upgradev1.UpgradePhasePostCheck:
        return c.handlePostCheck(ctx, clusterVersion)
    case upgradev1.UpgradePhaseFailed:
        return c.handleFailed(ctx, clusterVersion)
    case upgradev1.UpgradePhaseRollback:
        return c.handleRollback(ctx, clusterVersion)
    default:
        return nil
    }
}

// handleIdle 处理空闲状态
func (c *UpgradeCoordinator) handleIdle(ctx context.Context, cv *upgradev1.ClusterVersion) error {
    // 检测是否需要升级
    needUpgrade, plan, err := c.detector.Detect(ctx, cv)
    if err != nil {
        return err
    }
    
    if !needUpgrade {
        return nil
    }
    
    // 更新状态为 PreCheck
    cv.Status.Phase = upgradev1.UpgradePhasePreCheck
    cv.Status.StartTime = &metav1.Time{Time: time.Now()}
    
    return c.Status().Update(ctx, cv)
}

// handlePreCheck 处理前置检查
func (c *UpgradeCoordinator) handlePreCheck(ctx context.Context, cv *upgradev1.ClusterVersion) error {
    // 执行前置检查
    result, err := c.checker.RunPreCheck(ctx, cv)
    if err != nil {
        return err
    }
    
    if !result.Passed {
        // 检查失败
        cv.Status.Phase = upgradev1.UpgradePhaseFailed
        cv.Status.Conditions = append(cv.Status.Conditions, metav1.Condition{
            Type:    "PreCheckFailed",
            Status:  metav1.ConditionTrue,
            Reason:  "PreCheckFailed",
            Message: fmt.Sprintf("Pre-check failed: %v", result.Errors),
        })
        return c.Status().Update(ctx, cv)
    }
    
    // 检查通过，进入准备阶段
    cv.Status.Phase = upgradev1.UpgradePhasePreparing
    return c.Status().Update(ctx, cv)
}

// handlePreparing 处理准备阶段
func (c *UpgradeCoordinator) handlePreparing(ctx context.Context, cv *upgradev1.ClusterVersion) error {
    // 准备升级资源
    if err := c.executor.Prepare(ctx, cv); err != nil {
        return err
    }
    
    // 进入升级阶段
    cv.Status.Phase = upgradev1.UpgradePhaseUpgrading
    return c.Status().Update(ctx, cv)
}

// handleUpgrading 处理升级阶段
func (c *UpgradeCoordinator) handleUpgrading(ctx context.Context, cv *upgradev1.ClusterVersion) error {
    // 执行升级
    completed, err := c.executor.Execute(ctx, cv)
    if err != nil {
        if cv.Spec.Strategy.RollbackOnFailure {
            cv.Status.Phase = upgradev1.UpgradePhaseRollback
        } else {
            cv.Status.Phase = upgradev1.UpgradePhaseFailed
        }
        return c.Status().Update(ctx, cv)
    }
    
    if completed {
        // 升级完成，进入后置检查
        cv.Status.Phase = upgradev1.UpgradePhasePostCheck
        return c.Status().Update(ctx, cv)
    }
    
    // 继续升级
    return nil
}

// handlePostCheck 处理后置检查
func (c *UpgradeCoordinator) handlePostCheck(ctx context.Context, cv *upgradev1.ClusterVersion) error {
    // 执行后置检查
    result, err := c.checker.RunPostCheck(ctx, cv)
    if err != nil {
        return err
    }
    
    if !result.Passed {
        if cv.Spec.Strategy.RollbackOnFailure {
            cv.Status.Phase = upgradev1.UpgradePhaseRollback
        } else {
            cv.Status.Phase = upgradev1.UpgradePhaseFailed
        }
        return c.Status().Update(ctx, cv)
    }
    
    // 升级成功
    cv.Status.Phase = upgradev1.UpgradePhaseSuccess
    cv.Status.CompletionTime = &metav1.Time{Time: time.Now()}
    cv.Status.CurrentVersion = cv.Spec.DesiredVersion
    
    // 记录历史
    cv.Status.History = append(cv.Status.History, upgradev1.UpgradeHistory{
        Version:        cv.Spec.DesiredVersion,
        State:          "Completed",
        StartTime:      *cv.Status.StartTime,
        CompletionTime: cv.Status.CompletionTime,
    })
    
    return c.Status().Update(ctx, cv)
}

// handleFailed 处理失败状态
func (c *UpgradeCoordinator) handleFailed(ctx context.Context, cv *upgradev1.ClusterVersion) error {
    // 根据策略决定是否重试
    if cv.Spec.Strategy.PauseOnFailure {
        cv.Status.Phase = upgradev1.UpgradePhasePaused
        return c.Status().Update(ctx, cv)
    }
    
    // 可以设置自动重试逻辑
    return nil
}

// handleRollback 处理回滚
func (c *UpgradeCoordinator) handleRollback(ctx context.Context, cv *upgradev1.ClusterVersion) error {
    // 执行回滚
    if err := c.rollback.Execute(ctx, cv); err != nil {
        return err
    }
    
    // 回滚完成
    cv.Status.Phase = upgradev1.UpgradePhaseFailed
    return c.Status().Update(ctx, cv)
}
```

```go
// Register 注册 Provider
func (r *Registry) Register(p Provider) {
    r.mu.Lock()
    defer r.mu.Unlock()
    r.providers[p.Name()] = p
}

// Get 获取 Provider
func (r *Registry) Get(name string) (Provider, error) {
    r.mu.RLock()
    defer r.mu.RUnlock()
    
    p, ok := r.providers[name]
    if !ok {
        return nil, fmt.Errorf("provider not found: %s", name)
    }
    return p, nil
}

// Detect 检测并返回匹配的 Provider
func (r *Registry) Detect(ctx context.Context, node NodeInfo) (Provider, error) {
    r.mu.RLock()
    defer r.mu.RUnlock()
    
    for _, p := range r.providers {
        matched, err := p.Detect(ctx, node)
        if err != nil {
            continue
        }
        if matched {
            return p, nil
        }
    }
    
    return nil, fmt.Errorf("no matching provider found for node: %s", node.IP)
}

// List 列出所有 Provider
func (r *Registry) List() []string {
    r.mu.RLock()
    defer r.mu.RUnlock()
    
    names := make([]string, 0, len(r.providers))
    for name := range r.providers {
        names = append(names, name)
    }
    return names
}
```

```go
// pkg/statemachine/machine.go
type ClusterStateMachine struct {
    client       client.Client
    cluster      *bkev1beta1.BKECluster
    states       map[ClusterPhase]State
    currentState ClusterPhase
    phaseContext *phaseframe.PhaseContext
}

func NewClusterStateMachine(cluster *bkev1beta1.BKECluster, ctx *phaseframe.PhaseContext, c client.Client) *ClusterStateMachine

func (sm *ClusterStateMachine) Reconcile(ctx context.Context) (ctrl.Result, error) {
    state := sm.states[sm.currentState]
    nextPhase, err := state.Execute(ctx, sm)
    if err != nil {
        sm.transitionTo(PhaseFailed)
        return ctrl.Result{}, err
    }
    if nextPhase != sm.currentState {
        if err := sm.transition(ctx, nextPhase); err != nil {
            return ctrl.Result{}, err
        }
    }
    return sm.getResult()
}

func (sm *ClusterStateMachine) transition(ctx context.Context, targetPhase ClusterPhase) error {
    if !IsTransitionAllowed(sm.currentState, targetPhase) {
        return fmt.Errorf("invalid transition: %s -> %s", sm.currentState, targetPhase)
    }
    currentState := sm.states[sm.currentState]
    targetState := sm.states[targetPhase]
    if err := currentState.Exit(ctx, sm); err != nil { return err }
    if err := targetState.Enter(ctx, sm); err != nil { return err }
    sm.currentState = targetPhase
    sm.cluster.Status.Phase = confv1beta1.BKEClusterPhase(targetPhase)
    return sm.client.Status().Update(ctx, sm.cluster)
}
```

```go
    }

func (s *ProvisioningState) Name() ClusterPhase { return PhaseProvisioning }

func (s *ProvisioningState) Enter(ctx context.Context, sm *ClusterStateMachine) error {
    // 初始化部署阶段状态
    return nil
}

func (s *ProvisioningState) Execute(ctx context.Context, sm *ClusterStateMachine) (ClusterPhase, error) {
    // 依次执行 DeployPhases（复用现有 Phase 接口）
    for _, phase := range s.phases {
        if phase.NeedExecute(sm.cluster, sm.cluster) {
            if err := phase.ExecutePreHook(); err != nil {
                return PhaseFailed, err
            }
            result, err := phase.Execute()
            if err != nil {
                phase.ExecutePostHook(err)
                return PhaseFailed, err
            }
            phase.ExecutePostHook(nil)
            if result.RequeueAfter > 0 {
                return PhaseProvisioning, nil // 继续当前状态
            }
        }
    }
    return PhaseRunning, nil // 部署完成，转换到 Running
}

func (s *ProvisioningState) Exit(ctx context.Context, sm *ClusterStateMachine) error {
    return nil
}

func (s *ProvisioningState) CanTransitionTo(target ClusterPhase) bool {
    return IsTransitionAllowed(PhaseProvisioning, target)
```

```go
// controllers/upgrade/clusterversion_controller.go
type ClusterVersionReconciler struct {
    client.Client
    Scheme      *runtime.Scheme
    Recorder    record.EventRecorder
    Detector    *UpgradeDetector
    Checker     *UpgradeChecker
    Coordinator *UpgradeCoordinator
}

func (r *ClusterVersionReconciler) Reconcile(ctx context.Context, req ctrl.Request) (ctrl.Result, error) {
    cv := &upgradev1.ClusterVersion{}
    if err := r.Get(ctx, req.NamespacedName, cv); err != nil {
        return ctrl.Result{}, client.IgnoreNotFound(err)
    }

    switch cv.Status.Phase {
    case "", UpgradePhaseIdle:
        return r.handleIdle(ctx, cv)
    case UpgradePhasePreCheck:
        return r.handlePreCheck(ctx, cv)
    case UpgradePhaseUpgrading:
        return r.handleUpgrading(ctx, cv)
    case UpgradePhasePostCheck:
        return r.handlePostCheck(ctx, cv)
    case UpgradePhaseFailed:
        return r.handleFailed(ctx, cv)
    case UpgradePhaseRollback:
        return r.handleRollback(ctx, cv)
    case UpgradePhasePaused:
        return r.handlePaused(ctx, cv)
    }
    return ctrl.Result{}, nil
}
```
